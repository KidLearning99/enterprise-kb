## How to Use This Registry

1. Every time you save a new document (LinkedIn post, company KB, vendor article), add a row here
2. Set **Status** to `To Process` initially
3. After extracting questions and creating KB entries, set Status to `Processed`
4. In your KB entries, reference source IDs in the `Sources` metadata field

## Status Legend

| Status | Meaning |
|---|---|
| `To Process` | Document saved but not yet extracted into KB entries |
| `In Progress` | Partially processed — some questions extracted |
| `Processed` | All extractable knowledge has been added to KB entries |
| `Archive` | Outdated or superseded by newer sources |
